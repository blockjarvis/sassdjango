Website feito em Django com Integração de API e Preços de Metais Preciosos em Tempo Real

Sobre

Este website foi criado para apresentar minhas habilidades usando o framework Django, Python, SASS, JS, HTML, CSS e integração com API. Devido a limitações de tempo, eu só fui capaz de incluir um aplicativo, mas com mais tempo, planejo adicionar um sistema de autenticação e outros aplicativos, como um blog, sistema de compras e página de contato. Para construir este site, eu usei duas APIs. A primeira API, fornecida pela Metal Price API, é usada no cabeçalho para exibir preços de metais preciosos em tempo real. Eu programei para mostrar preços em onças para três metais diferentes em GBP e EUR. A segunda API, fornecida pelo TradingView, é usada para mostrar o gráfico GOLD/GBP. Além disso, o site oferece uma segunda opção de idioma (Ingles).
